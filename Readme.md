##项目结构
* src 
    * main 代码
        * java java代码
            * domain 数据库实体类
                * Lanmu 栏目名(院系概况，专业建设等)
                * News 新闻类(用于封装新闻通知)
            * util 工具类
            * web web相关的类(如servlet)
                * Servlet 
                    * ShouyeServlet 显示首页信息
                    * NewServlet 显示新闻消息       
        * resources 资源文件
        * webapp 界面文件
